## 3.Dataset
We decided to use Wikipedia comments that was used in [Toxic Comment Classification Challenge](https://www.kaggle.com/c/jigsaw-toxic-comment-classification-challenge/data). Dataset has the following labels:

- toxic

- severe_toxic

- obscene

- threat

- insult

- identity_hate
